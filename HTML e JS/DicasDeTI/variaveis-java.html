Variáveis de Ambiente Java

Podemos colocar na variável de ambiente do sistema, os arquivos do JRE e JDK para que assim, possamos executar e compilar bytecodes Java através da linha de comando.

Windows

Após a instalação do JDK, vá em computador, propriedades do sistema e configurações avançadas do sistema, em avançado vá em variáveis de ambiente.

Nas variáveis de usuário, veja se em PATH está com %JAVA_HOME%\bin, senão adicione ali.

Nas variáveis do sistema, crie a variável JAVA_HOME com o caminho do JDK (por exemplo C:\Program Files\Java\jdk1.8.0_181) e JRE_HOME com o caminho do JRE (por exemplo C:\Program Files\Java\jre1.8.0_181).

Linux

Após a instalação do pacote Java, vá na pasta dele e digite esses comandos:


sudo ln -s /opt/jdk1.8.0_181/java /usr/bin/java

sudo ln -s /opt/jdk1.8.0_181/javac /usr/bin/javac

Isso criará um atalho no diretório /usr/bin, que por padrão está na variável de ambiente do Linux, e permitirá o acesso pelo terminal.

PS: Deve ser colocado o caminho daonde foi instalado o JDK, caso seja outro.
