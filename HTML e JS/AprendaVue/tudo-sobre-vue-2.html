    Tudo sobre Vue JS Parte 2

    Diretiva v-bind - Atribuindo valor em class e style

    Crie dentro da DIV esses botões:

    
<button class="btn">Limpar</button>
<button class="btn">Enviar</button>

    Adicione a linha do Bootstrap também, no head:

    
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">

    No VUE, para colocarmos as classes do Bootstrap, faça assim:

    
var app = new Vue({
    el: "#app",
    data: {
        btnClassLimpar: "btn-danger",
        btnClassEnviar: "btn-primary"
    }
});

    Coloque o v-bind:class dos botões enviar e limpar, usando suas variáveis, assim:

    
v-bind:class="btnClassLimpar"

    Podemos usar com objetos e arrays também:

    
var app = new Vue({
    el: "#app",
    data: {
        btnClassLimpar: {
            "btn-danger": true,
            "btn-sm": true
        },
        btnClassEnviar: ["btn-primary", "btn-lg"]
    }
});

    Ou unir ambos:

    
var app = new Vue({
    el: "#app",
    data: {
        btnClassLimpar: {
            "btn-danger": true,
            "btn-sm": true
        },
        btnClassEnviar: ["btn-primary", {
                        "btn-lg": false
                    }]
    }
});

    Coloque agora a diretiva v-bind:style em ambos os botões, com as variáveis btnStyleLimpar e btnStyleEnviar.

    Coloque no Vue, dentro de data, essas variáveis:

    
btnStyleLimpar: "font-size: 50pt",
btnStyleEnviar: "font-size: 10pt"

    Como objetos:

    
btnStyleLimpar: {
    "font-size": "50px",
    "text-transform": "lowercase"
},
btnStyleEnviar: {
    "font-size": "10px",
    "text-transform": "uppercase"
}

    PS: Podemos remover o v-bind nos elementos HTML, deixando apenas os dois pontos, algo como :class e :style.

    Diretiva v-on - Manipulando Eventos

    Para usarmos os métodos do Vue, fazemos usando o campo methods, assim:

    
var app = new Vue({
    el: "#app",
    data: {
        
    },

    methods: { // É tipo uma classe

    }
});

    Dentro do methods, faça essa função simples:

    
methods: {
    enviar() {
        alert("Enviado!");
    }
}

    No botão de enviar dentro da DIV, coloque essa diretiva:

    
v-on:click="enviar()"

    Podemos ter combinação de eventos também, por exemplo, num formulário:

    Faça esse formulário dentro da DIV:

    
<form method="post" action="https://www.google.com.br">
    <input type="submit" value="Enviar" class="btn btn-primary" v-on:click="enviar()"/>
</form>

    Só que no caso acima, ele vai exibir o alerta e mandar pro Google. Para evitar esse redirecionamento automático, coloque v-on:click.prevent.stop="enviar()"

    PS: Podemos remover o v-on nos elementos HTML, deixando apenas o arroba, algo como @click.
