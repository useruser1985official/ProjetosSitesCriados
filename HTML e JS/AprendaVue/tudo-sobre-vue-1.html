    Tudo sobre Vue JS Parte 1

    Introdução, Instalação e Aprendendo sobre Reatividade

    Crie um documento HTML e adicione esse link na tag script: https://unpkg.com/vue

    Crie uma div com a id "app", e numa nova tag script, crie o objeto dessa forma:

    
var app = new Vue({

});

    Adicione alguns elementos assim:

    
var app = new Vue({
    el: "#app", // ID do elemento pai
    data: { // Onde estarão nossas variáveis
        titulo: "Aula 01 de Vue JS"
    }
});

    Para escrever o elemento, vá na div criada e coloque dentro dela, a variável entre duas chaves, assim:

    
{{titulo}}

    PS: Podemos formatar usando outras tags, como o H1.

    Lembrando que o código criado só vai funcionar dentro da div criada, pelo fato da ID ter sido definida no código.

    Tudo que produzir no Vue é reativo, sempre que for alterado no objeto o conteúdo ele será alterado no documento. Inclusive se abrirmos o console do navegador e fizermos uma nova atribuição, ele alterará. Digite no console esse comando:

    
app.titulo = "Novo Título"

    O que são Diretivas no Vue JS

    As diretivas são algo que dirige algo para que ela faça alguma tarefa.

    As diretivas são colocadas dentro das tags HTML com as quais trabalhamos. Ele tem várias diretivas por padrão e podemos criar nossas próprias também.

    As diretivas padrões começam com "v-", como v-model, v-if e v-for.

    Diretivas mais Utilizadas

    Vamos supor que queremos imprimir uma variável quando o status for verdadeiro, podemos fazer isso com o seguinte código:

    
var app = new Vue({
    el: "#app",
    data: {
        status: false,
        titulo: "Aula 01 de Vue JS"
    }
});

    E no HTML, colocamos a diretiva v-if, dessa forma:

    
<div id="app">
    <h1 v-if="status">{{titulo}}</h1>
</div>

    Ela não imprimirá se o status for falso, se trocarmos para verdadeiro ele exibirá.

    PS: Da mesma forma, na diretiva if podemos colocar uma condição, como num == 10.

    A diretiva v-show funciona de forma parecida, a diferença é que com v-if, caso seja falso, ele excluirá o texto do código-fonte (olhando pelo inspetor), e com v-show ele apenas adicionará um estilo CSS com display none.

    Coloque dentro da nossa div um texto qualquer usando o v-show e outro o v-if para vermos aa diferença.

    O v-show seria para, por exemplo, um menu abre-e-fecha ou mostrar-ocultar. O v-if seria mais para checar uma variável vinda de uma API ou do back-end (como do PHP ou Node).

    A diretiva v-else é utilizada junto com a v-if, que exibe algo caso seja falso. Veja um exemplo de uso:

    
<div id="app">
    <h1 v-if="status">{{titulo}}</h1>
    <h1 v-else>Título não Encontrado</h1>
</div>

    Deixe o código assim:

    
var app = new Vue({
    el: "#app",
    data: {
        usuario: { // Usuário é um objeto
            id: 1,
            nome: "Fulano",
            profissao: "Ator"
        }
    }
});

    E no HTML, faça assim:

    
<div id="app">
    <div v-if="usuario.id == 1">
        ID: {{usuario.id}}<br/>
        Nome: {{usuario.nome}}<br/>
        Profissão: {{usuario.profissao}}<br/>
    </div>
    <div v-else>
        Nenhum Usuário com o ID especificado.<br/>
    </div>
</div>

    Diretiva v-for e suas Peculiariedades

    Deixe o código assim, com um array de objetos:

    
var app = new Vue({
    el: "#app",
    data: {
        linguagens: [{nome: "Javascript"},
            {nome: "PHP"},
            {nome: "C#"},
            {nome: "Java"},
            {nome: "Python"}]
    }
});

    E para mostrarmos os itens através de um loop, colocamos a diretiva v-for assim:

    
<div id="app">
    <ul>
        <li v-for="i of linguagens">{{i}}</li>
    </ul>
</div>

    Ele mostrará cada objeto do array em uma lista. Para pegar só o nome, use o atributo nome do objeto.

    PS: Para imprimir o index, coloque a diretiva assim:

    
<li v-for="i, n of linguagens">{{n}} - {{i.nome}}</li>

    Diretiva v-model - Interligando Formulário e Dados

    A diretiva v-model é utiliza em elementos de formulários, como o inputt e o textarea. Crie um input text dentro da div com a diretiva v-model com a variável nome. E declare a variável no Javascript, assim>

    
var app = new Vue({
    el: "#app",
    data: {
        nome: ""
    }
});

    Tente exibir a variável nome usando {{nome}} no HTML, dentro da div id app. Tudo que for escrito no input aparecerá no HTML. Isso porque o Vue é reativo.

    Crie esse formulário dentro da DIV:

    
<form method="get">
    <p><label for="cNome">Nome: </label><input type="text" name="tNome" id="cNome" v-model="nome"/></p>
    <p>
        Sexo:
        <input type="radio" name="tSex" id="cMasc" value="M" v-model="sexo" checked/><label for="cMasc">Masculino</label>
        <input type="radio" name="tSex" id="cFem" value="F" v-model="sexo"/><label for="cFem">Feminino</label>
    </p>
    <p>
        <label for="cNac">Nacionalidade: </label>
        <select name="tNac" id="cNac" v-model="nacionalidade">
            <option>Brasileiro</option>
            <option>Estrangeiro</option>
        </select>
    </p>
    <p>
        Interesses:
        <input type="checkbox" name="tEsp" id="cEsp" value="Esportes" v-model="interesses"/><label for="cEsp">Esportes</label>
        <input type="checkbox" name="tCin" id="cCin" value="Cinema" v-model="interesses"/><label for="cCin">Cinema</label>
        <input type="checkbox" name="tGas" id="cGas" value="Gastronomia" v-model="interesses"/><label for="cGas">Gastronomia</label>
    </p>
</form>

    No Vue, faça isso:

    
var app = new Vue({
    el: "#app",
    data: {
        nome: "",
        sexo: "",
        nacionalidade: "",
        interesses: []
    }
});

    Depois é só chamar as variáveis dentro da div.

    Diretiva v-bind - Atribuindo Valor as Tags HTML

    Delete tudo de dentro da div e coloque essa tag img:

    
<img v-bind:src="imagem" v-bind:alt="alt"/>

    E no Vue, faça assim:

    
var app = new Vue({
    el: "#app",
    data: {
        imagem: "imagens/desktop-e-celular.png",
        alt: "Imagem do Vue"
    }
});

    Da mesma forma, podemos fazer o mesmo com qualquer atributo do HTML, como title, id, class, etc.

    PS: Dessa forma, podemos colocar atributos sem precisar do objeto Vue, assim:

    
<img v-bind:src="'imagens/desktop-e-celular.png'" v-bind:alt="'Imagem do Vue'"/>

    Agora, crie esse estilo no CSS:

    
.red {
    color: red;
}

.green {
    color: green;
}

    Coloque dentro da DIV esse código com a diretiva:

    
<p v-bind:class="classe">Esse texto será vermelho ou verde!</p>

    E no Vue, coloque isso:

    
var app = new Vue({
    el: "#app",
    data: {
        classe: "green"
    }
});
