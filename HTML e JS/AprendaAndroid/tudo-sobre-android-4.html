    Exemplo de uso de SeekBar (Slider)

    Vá em Widgets e coloque um SeekBar com a id sliUrg e um labem com a id lblUrg, dentro de layouts.

    Coloque isso na função onCreate:

    
sliUrg = (SeekBar)findViewById(R.id.sliUrg); // Declare o atributo do tipo SeekBar
lblUrg = (TextView)findViewById(R.id.lblUrg); // Idem, tipo TextView

sliUrg.setOnSeekBarChangeListener(new SeekBar.OnSeekBarChangeListener() {
    @Override
    public void onProgressChanged(SeekBar seek, int i, boolean b) {
       lblUrg.setText(Integer.toString(i));
    }

    @Override
    public void onStartTrackingTouch(SeekBar seek) {

    }

    @Override
    public void onStopTrackingTouch(SeekBar seek) {

    }
});

    Podemos definir o valor máximo em all attributes.
