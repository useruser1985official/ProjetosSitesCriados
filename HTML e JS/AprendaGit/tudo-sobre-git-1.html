    Configurações Iniciais do Git

    Para configurar um perfil para nossos repositórios, digite esses comandos:

    
git config --global user.name "Nome do Usuário"

git config --global user.email "nomedousuario@servidor.com"

    Para verificar se o usuário realmente foi inserido, digite git config --list.

    Obviamente, o usuário já deve existir no site do Github ou outro site baseado em Git.

    Os arquivo de configuração ficam em C:\Users\usuario\ no Windows e em /home/usuario no Linux. O arquivo principal de configuração é o .gitconfig, que fica oculto no sistema.

    Para deslogar, caso precise, rode esses comandos:

    
git config --global --unset user.name

git config --global --unset user.email

    Para verificar se o usuário foi retirado, rode o comando list de novo.
